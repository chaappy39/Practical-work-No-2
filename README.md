## Удаление латинских букв:
### При выделении текста и нажатии на кнопку "Удалить латинские буквы", выделенный фрагмент текста будет обработан — из него удалятся все латинские буквы (как заглавные, так и строчные).

## Удаление последнего пробела:
### Если выделить текст и нажать на кнопку "Удалить последний пробел", в выделенном фрагменте текста будет удален последний символ пробела (если он есть).
